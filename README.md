# 📈 StockSense - ML-Powered Stock Analytics & Prediction

StockSense is a professional full-stack **MERN (MongoDB, Express, React, Node.js)** web application modeled with a premium, Zerodha-inspired interface. It tracks real-time market metrics and uses mathematical models (Linear Regression and Holt-Linear Trend) to predict future stock prices directly from historical database entries.

## ✨ Features

1. **🧠 Machine Learning Price Predictor:**
   * Run 7, 15, or 30-day forecast windows on assets.
   * Toggle between **Linear Regression** (best for long-term trendlines) and **Holt-Linear Double Exponential Smoothing** (best for momentum and trend-change detection).
   * Generates confidence scores ($R^2$) and calculates Min/Max forecast bounds.

2. **📊 Dynamic Charting & Real-Time Dark Mode:**
   * Interactive historical charts displaying closing prices and volumes.
   * Global light/dark theme swapper that automatically saves preferences in `localStorage`.
   * Dynamic Chart.js grid line and text color updates using DOM mutation observers when switching themes.

3. **⭐ Watchlist Sidebar Filter:**
   * Star your favorite stock tickers on the dashboard.
   * Instantly filter the stocks catalogue sidebar to show either "All Stocks" or "Watchlist" selections using MongoDB queries.

4. **📈 Extended Market Metrics:**
   * Dynamic calculations of **52-Week High**, **52-Week Low**, and **Average Trading Volume** computed directly from database history.

5. **🔒 Secure Authentication & Settings:**
   * User registration and login protected with JSON Web Tokens (JWT) and Bcrypt password hashing.
   * Profile dashboard showing account stats (e.g., account level, join date, watchlist size).
   * Forms to change passwords (with validation) and update full names.

## 🚀 Getting Started (Local Setup)

Follow these steps to run the application on your computer:

### 1. Clone and Navigate
```bash
git clone https://github.com/Ayushsingh0128/StockSense.git
cd StockSense
```

### 2. Configure Backend
Create a `.env` file in the `/backend` folder:
```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
```

Install backend dependencies and seed the database:
```bash
cd backend
npm install
npm run seed      # Populates database with 365 days of OHLCV history
npm run dev       # Starts REST server on http://localhost:5000
```

### 3. Configure Frontend
Install frontend dependencies and start the Vite dev server:
```bash
cd ../frontend
npm install
npm run dev       # Starts UI on http://localhost:5173
```
